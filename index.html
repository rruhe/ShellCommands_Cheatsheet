Cheatsheet for Shell/Bash Commands

What you need in a terminal. List of commands available:

  • ls
  • cd
  • cp
  • mv
  • mkdir
  • touch

Command:ls

List of directories (display the contents of a directory)

You can use this command with parameters

Command:ls -l

List files and folders with details

Command:ls -a

List also hidden files and folders

Command:ls -la

List also hidden files and folders with details

Command:cd

cd folder

goto the folder

Command:cd

cd folder/folder/folder

goto the folder and subfolders

Command:cd

cd ..

Go back to the parent folder

Command:cd

cd ../../../

Go back to the parent folders

Command:cd

cd /

Goto the root folder of this filesystem

Command:cp

copy file file

Copy a file in the same folder

Command:cp

cp -r folder(subfolders) folder

Copy the folder recursive with subfolders and files to the destination

Command:mv

mv file folder

The command mv (move) is used to move files and folders

Command:mkdir

mkdir theNewfolder

The command is for creating new folders

Command:touch

touch index.html

The command is for creating new file
